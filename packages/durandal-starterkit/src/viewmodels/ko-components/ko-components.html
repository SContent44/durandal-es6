Knockout components

Here are 3 nested components. Grandparent -> Parent -> Child. If you open up your network tab (and turn on throttling to make it a bit clearer) when you load this page you will see:

  1. The pages ViewModel load
  2. The child component load (with a descriptive name) and the grandparent component load at about the same time
  3. The parent component load

This is a showcase of how the application has control of when to lazy-load modules (including components which are "loaded asynchronously")
